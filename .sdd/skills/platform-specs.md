# Skill — Platform Specs

## Per-channel rules

| Platform | Char limit | Format | Hashtag rule | Tone |
|---|---|---|---|---|
| Zalo | 1000 | plain VI, link cuối | 0 | thân thiện |
| Facebook DM | 2000 | plain VI, link inline OK | 0 | formal hơn |
| TikTok DM | 500 | rất ngắn, có emoji | 2–4 | trẻ |
| TikTok comment | 150 | reply + mời DM | 0 | trẻ |
| Instagram DM | 1000 | visual hint + ngắn | 2 | visual |
| Instagram caption | 2200 | hook đầu + line breaks | 5–10 | visual |
| YouTube comment | 500 | info + link | 0 | informative |

## Banned phrases
- "Đặt cọc ngay" (manipulation)
- "Rẻ nhất thị trường" (compliance)
- Slang phản cảm: xem `content-copywriting.md`

## Posting golden hours
- Zalo: 11:30–13:00, 19:30–22:30
- Facebook: 12:00–13:00, 20:00–22:00
- TikTok: 06:00–09:00, 19:00–23:00
- Instagram: 11:00–14:00, 17:00–21:00
- YouTube: upload 18:00–21:00 thứ 6
